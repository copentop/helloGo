## Go 易错点

### slice append
slice 使用append时，但容量不够时会创建新slice并将久但内容拷贝到新但slice上，同时返回新slice的指针。

易错在于容量不够时，并没有返回新的slice指针。

```
package main
import "fmt"

func Pingpong(s []int ) {
    s = append(s, 3)
}
func main() {
    // 容量只有零个
    s := make([] int, 0)
    fmt.Println(s) // []
    // 容量不够，创建新slice，但没有返回新指针，还是久指针
    Pingpong(s)
    fmt.Println(s) // 
}


```
